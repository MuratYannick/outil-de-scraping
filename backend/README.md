# Backend - API Node.js

À compléter selon les technologies définies par le chef de projet.

## Structure

- `src/` - Code source de l'API
- `config/` - Configuration
- `models/` - Modèles de données
- `routes/` - Routes API
- `controllers/` - Logique métier
- `services/` - Services (scraping, etc.)
- `middlewares/` - Middlewares

## À définir

- Version de Node.js
- Framework Web (Express, Fastify, etc.)
- ORM/Driver Base de données
- Librairie de scraping (Playwright, Puppeteer, etc.)
